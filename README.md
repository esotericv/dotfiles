# Packages used

```
## Bar
waybar 

## Hypr ecosystem
hyprshot 

hyprpicker

hyprlock

rose-pine-hyprcursor - hyprcursor theme

## Essentials/compatibility 
archlinux-xdg-menu - used for file associations in dolphin

xdg-desktop-portal-hyprland

network-manager-applet - for networking

pavucontrol - volume management

dunst - notifications

swww - wallpaper daemon

mpd - music player daemon

mpdscribble - (optional) last.fm scrobbling for mpd

uwsm - universal wayland session manager, allows systemd services to run properly

qt6ct-kde - configure qt6 themes (kde variant is patched to work with kde apps)

## Userspace apps 
yazi - tui file explorer

rmpc or ncmpcpp or cmus - tui music player

fuzzel - app launcher

wallust - color scheme generator

## Fun terminal animations
cava - visualizer
cbonsai - bonsai tree
lavat - lava lamp (lavat-git)
astroterm - view your local sky ( or anyhwere you want) from your terminal
